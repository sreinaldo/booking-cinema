Booking Cinema
===========================
1. Clone the repo

   ```
   $ git clone 
   $ cd booking-cinema
   ```

2. Install dependencies

   ```
   $ bundle install
   ```

3. Run specs

   ```
   $ rspec spec/api
   ... 0 failures
   ```

4. Run app

   ```
   $ rails s

   ```

4. API documentation

   ```
   http://localhost:3000/api/swagger_doc

   ```

# Dependencies:
* Ruby version: 2.5.1
* Rails version: 5.2.4
